enum tch_lplvl

An enum for defining low power level

Enumerator
LP_LEVEL0 

CPU Idle : CPU clock source is gated

LP_LEVEL1 

CPU Sleep : CPU clock is turned off, contents in SRAM are kept

LP_LEVEL2 

CPU Deep Sleep : CPU clock is turned off, contents in SRAM are lost

DMA HAL Object

DMA HAL Object, which is not directly accessible to user program. it's been recommended that DMA is accessed by another HAL Interface

tch_rtc

RTC HAL Object

RTC HAL Object, which is not directly accessible to user program. RTC is abstracted to System Timer, which keeps track of real time information of system and also deals with timed scheduling of task
