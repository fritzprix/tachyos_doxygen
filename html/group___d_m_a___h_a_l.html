Created on: 2014. 6. 15. Author: innocentevil

This is HAL Interface for DMA Module. Interface itself is desinged agnostic to Platform H/W though DMA has H/W dependencies in following reason.

  • Number of DMA Stream and channel which can be used with specific H/W module is fixed by H/W Implementation. So if DMA HAL is freely accessible to user, it can be error prone unless user carefully investigates platform H/W datasheet and this can make user feel frustrating port their tachyos application code
